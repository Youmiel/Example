# 草方块 （由另一位匿名用户翻译）

**作者:** 匿名

**发布日期:** 2024-12-16

**许可证:** CC0 1.0 Universal

**简介**

某人不小心把他的红石电路保存成了一个草方块。

![A Grass Block](media/grass-block.png)

**数据**

碰撞箱 (以方块像素为单位): 16 x 16 x 16 (完整方块)

**技术解释**

> Grass can spread to nearby dirt blocks, but not coarse dirt or rooted dirt. 
> 
> When grass is random ticked, it spreads by checking 4 blocks within range and converts any dirt blocks selected into grass.
>
> -- Minecraft Wiki (EN): [Grass Block](https://minecraft.wiki/w/Grass_Block)

**优点**

- 可以用来获取种子和花朵。
- 动物可以在上面生成。
- 草方块自然的外观适合园林绿化。

**缺点**

- 需要用精准采集才能获得。
- 如果在其上方放置不透明方块，或者周围变得过于黑暗，它将退化成泥土。
