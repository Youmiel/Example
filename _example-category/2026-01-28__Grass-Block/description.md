# A Grass Block

**Author(s):** Anonymous

**Release Date:** 2024-12-16

**License:** CC0 1.0 Universal

**Description**

Someone accidentally saved a grass block instead of his contraption. Here it is.

![A Grass Block](media/grass-block.png)

**Statistics**

Collision box (in pixels): 16 x 16 x 16 (full block)

**Technical Details**

> Grass can spread to nearby dirt blocks, but not coarse dirt or rooted dirt. 
> 
> When grass is random ticked, it spreads by checking 4 blocks within range and converts any dirt blocks selected into grass.
>
> -- Minecraft Wiki: [Grass Block](https://minecraft.wiki/w/Grass_Block)

**Pros**

- It can be used to obtain seeds and flowers.
- Animals can spawn on it.
- Its natural looking is suitable for landscaping.

**Cons**

- It requires silk touch to obtain.
- It will turn into dirt if an opaque block is placed on top of it or the surroundings getting too dark.
